Vorgaben für die Verwendung des Musters Service Facade to WebSphere MQ: request-response (Servicefassade für WebSphere MQ: Anforderung-Antwort)

Wenn Sie dieses Muster verwenden, gelten eine Reihe von Vorgaben.

Sie können das Muster Service Facade to WebSphere MQ: request-response (Servicefassade für WebSphere MQ: Anforderung-Antwort) nur verwenden, wenn alle der folgenden Vorgaben erfüllt sind:

  • Die Anforderungen müssen als SOAP-Nachrichten über HTTP eingehen.
  • Die anfordernden Anwendungen müssen einen Interaktionsstil mit synchronen Anforderungen und Antworten unterstützen.
  • Die Provideranwendungen müssen Anforderungen innerhalb eines für eine synchrone Interaktion angemessenen Zeitrahmens beantworten können.
  • Die Provideranwendungen müssen Anforderungen über WebSphere MQ empfangen und die Antworten über WebSphere MQ zurückgeben können.
  • Die Provideranwendungen müssen das Standardverhalten zum Kopieren der Nachrichten-ID in die Korrelations-ID unterstützen.
  • Die WSDLs, die die Brokerschnittstelle definieren, weisen nur einen Porttyp und nur eine Bindung auf.
  • Die WSDLs müssen die Eigenschaften für den SOAPInput-Knoten definieren, der in der WSDL der höchsten Ebene definiert ist. Einige komplexe, verschachtelte WSDLs können daher nicht automatisch von diesem Muster verarbeitet werden.

    Wenn die verwendeten WSDLs diese Bedingungen nicht erfüllen, kann das Muster den SOAPInput-Knoten nicht konfigurieren. Die Konfiguration des SOAPInput-Knotens wird in diesem Fall als erforderliche Task nach der Generierung der Musterinstanz aufgeführt.

  • Die Provideranwendung unterstützt WebSphere MQ-XML-Nachrichten in einem Format, das als Hauptteil einer SOAP-Anforderung oder SOAP-Antwort verwendet werden kann. Sie können den untergeordneten Anforderungsprozessor-Nachrichtenfluss aber auch so anpassen, dass das Format für Nicht-XML-Nachrichten umgesetzt wird.
  • Wenn eine Protokollierung erforderlich ist, wird davon ausgegangen, dass das von diesem Muster verwendete Nachrichtenformat Ihren Anforderungen entspricht.
