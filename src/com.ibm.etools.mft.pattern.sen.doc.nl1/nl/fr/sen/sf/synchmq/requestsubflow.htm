Flux secondaire Traitement de demande pour le modèle Façade de service sur WebSphere MQ : question-réponse

Par défaut, ce flux secondaire n'a pas d'impact. Lorsque le modèle Façade de service pour WebSphere MQ : question-réponse est utilisé pour générer une instance de modèle, il contient uniquement un noeud Passthrough. Ce flux secondaire est inclus pour fournir un emplacement dans lequel une instance du modèle peut être personnalisée en utilisant des outils standard sans changer la structure du modèle.
