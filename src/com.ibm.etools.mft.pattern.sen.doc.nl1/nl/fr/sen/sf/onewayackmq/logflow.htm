Flux secondaire Journal pour le modèle Façade de service sur WebSphere MQ : unidirectionnel avec accusé de réception

Si le flux secondaire de consignation est inclus dans le flux Demande, la consignation des demandes est contrôlée par la propriété LoggingOn définie par l'utilisateur.

Les messages de journal sont écrits comme messages persistants WebSphere MQ dans une file d'attente WebSphere MQ. Ces messages de journal peuvent être traités séparément en fonction des pratiques standard de votre entreprise. Le journal des messages contient le message d'origine et les informations suivantes dans un en-tête MQRFH2 :

  • Horodatage
  • Nom du courtier
  • Nom du flux de messages
