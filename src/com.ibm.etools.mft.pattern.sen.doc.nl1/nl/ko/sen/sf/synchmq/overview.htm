WebSphere MQ에 서비스 퍼사드: 요청-응답 패턴

WebSphere MQ에 서비스 퍼사드: 요청-응답 패턴을 사용하면 WebSphere MQ를 통해서만 액세스할 수 있는 기능에 대한 웹 서비스 퍼사드를 제공할 수 있습니다. 이 패턴은 쉽게 업그레이드할 수 없는 WebSphere MQ 인터페이스를 포함한 기존 애플리케이션과 일반적으로 웹 서비스에 사용되는 동기 HTTP 프로토콜 간에 브릿지를 작성합니다.

제공자 애플리케이션이 XML 인터페이스를 제공하고 클라이언트 애플리케이션이 웹 서비스 호출을 지원하는 경우 이 패턴을 사용하십시오. 패턴을 확장하여 WebSphere MQ를 통해 비XML 인터페이스로 애플리케이션에 대한 서비스 퍼사드를 지원할 수 있습니다.

솔루션

솔루션은 서비스 시작점을 제공하는 메시지 플로우를 구현하는 것입니다. 서비스 요청을 수신할 때 응답 ID가 내부 큐에 저장되고 제공자 애플리케이션으로 요청이 전달됩니다. 제공자 애플리케이션으로부터 응답을 수신할 때는 응답 ID가 복구되고 제공자 응답을 포함한 웹 서비스 응답이 요청 애플리케이션으로 리턴됩니다.
