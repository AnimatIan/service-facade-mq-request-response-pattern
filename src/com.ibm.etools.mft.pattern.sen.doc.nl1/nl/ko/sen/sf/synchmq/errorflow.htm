WebSphere MQ에 서비스 퍼사드: 요청-응답 패턴의 오류 서브플로우

이 서브플로우는 오류 메시지가 기록되어야 하는지의 여부를 확인하며, 이는 사용자 정의 특성 ErrorLoggingOn에서 판별합니다.

  1. 오류 메시지를 포함하도록 패턴 매개변수를 구성하고 사용자 정의 특성 ErrorLoggingOn이 선택되면, 오류 메시지를 준비하고 오류 큐에 기록됩니다.

    예외 세부사항이 XML 메시지로 기록됩니다. 해당 세부사항은 다음과 같습니다.

    • 브로커 이름
    • 플로우 이름
    • 시간소인
    • 예외 데이터 요약

    예외 데이터 요약에는 메시지 플로우 레이블, 오류 번호, 텍스트 설명 및 추가 세부사항을 제공하는 삽입 목록이 포함되어 있습니다.

  2. 요청자에게 SOAP 결함을 리턴하는 예외가 발생합니다.
