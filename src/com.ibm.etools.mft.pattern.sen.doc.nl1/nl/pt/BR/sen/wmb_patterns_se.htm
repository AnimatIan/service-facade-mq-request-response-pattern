Ativação de Serviço

A Ativação de Serviço é uma das categorias comuns de padrões para soluções de conectividade.

O padrão de Ativação de Serviço encapsula a funcionalidade que não possui uma interface de serviço e apresenta esta funcionalidade através de uma interface orientada a serviço. Este padrão representa a transição da integração de aplicativos corporativos tradicionais nas Arquiteturas Orientadas a Serviços, permitindo que recursos existentes sejam reutilizados no novo estilo sem requerer mudança radical. Este padrão é usado para implementar um Barramento de Serviço Corporativo que fornece acesso aos serviços que são derivados da funcionalidade que é implementada usando diversas tecnologias na corporação e para apresentar uma única visualização consistente destes serviços aos consumidores de serviço.

Esse padrão também usa o Barramento de Serviço Corporativo do WebSphere para fornecer acesso aos serviços que alguns aplicativos não podem acessar diretamente.
