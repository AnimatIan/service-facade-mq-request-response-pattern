Subfluxo de Processamento de Pedidos para o padrão Service Facade para WebSphere MQ: Pedido-Resposta

Este subfluxo não possui, por padrão, nenhum impacto. Quando o padrão do Service Facade para WebSphere MQ: Pedido-Resposta é usado para gerar uma instância padrão, ele contém apenas um nó de Passagem. Este subfluxo é incluído para fornecer um local onde uma instância padrão pode ser customizada usando o conjunto de ferramentas padrão, sem alterar a estrutura do padrão.
