Habilitación de servicio

Habilitación de servicio es una de las categorías comunes de patrones de soluciones de conectividad.

El patrón Habilitación de servicio encapsula la funcionalidad que no tiene una interfaz de servicio y presenta esta funcionalidad a través de una interfaz orientada a servicios. Este patrón representa la transición de la integración de aplicaciones empresariales tradicional a las arquitecturas orientadas a servicios, lo que permite que los recursos existentes vuelvan a utilizarse en el nuevo estilo sin que ello requiera cambios radicales. Este patrón se utiliza para implementar un Bus de servicio empresarial que proporciona acceso a servicios que se derivan de funcionalidad que se implementa utilizando diversas tecnologías en la empresa y presentar una única vista coherente de estos servicios al consumidor del servicio.

Este patrón también utiliza el Bus de servicio empresaria de WebSphere para proporcionar acceso a los servicios a los que algunas aplicaciones no pueden acceder directamente.
