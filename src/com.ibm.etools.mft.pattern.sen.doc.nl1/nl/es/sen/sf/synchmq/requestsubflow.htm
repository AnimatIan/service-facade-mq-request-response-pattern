Subflujo Request Processing para el patrón Fachada de servicio a WebSphere MQ: solicitud-respuesta

De forma predeterminada, este subflujo no tiene ningún efecto. Cuando el patrón Fachada de servicio a WebSphere MQ: solicitud-respuesta se utiliza para generar una instancia de patrón, sólo contiene un nodo Passthrough. Este subflujo se incluye para proporcionar un lugar donde una instancia de patrón se puede personalizar utilizando herramientas estándar, sin cambiar la estructura del patrón.
